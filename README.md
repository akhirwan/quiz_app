# quiz_app

project demo is running in [Python Anywhere](https://akhirwan.pythonanywhere.com/)

instalation on local terminal (ubuntu):

- make sure you have MySQL and git is installed in your device
- create database in MySQL
- open terminal
- run script below

```
git clone https://github.com/akhirwan/quiz_app.git
```

- jump to project folder
```
cd quiz_app
```

- copy .env from .env.example
```
cp .env.example .env
```

- open `.env` and change the value to your credentials, **but do not change the keys**
- run this script
```
python3 app.py
```
